From map to spherical harmonics¶

From spherical harmonics to map¶

anafast(m[, lmax, mmax, iter, alm, ...]) Computes the power spectrum of an Healpix map.
alm2map(alm, nside[, lmax, mmax, pixwin, ...]) Computes an Healpix map given the alm.
alm2map_der1(alm, nside[, lmax, mmax]) Computes an Healpix map and its first derivatives given the alm.

Spherical harmonic transform tools¶

smoothing(m[, fwhm, sigma, degree, arcmin]) Smooth a map with a Gaussian symmetric beam.
smoothalm(alm[, fwhm, sigma, degree, ...]) Smooth alm with a Gaussian symmetric beam function in place.
alm2cl(alm[, mmax, nspec]) Compute the auto- and cross- spectra of the given alm’s (either 1 alm or 3 alm).
synalm(cls[, lmax, mmax]) Generate a set of alm given cl.
almxfl(alm, fl[, mmax, inplace]) Multiply alm by a function of l.
pixwin(nside[, pol]) Return the pixel window function for the given nside.
