healpy.sphtfunc.alm2cl¶

static sphtfunc.alm2cl(alm, mmax=-1, nspec=4)¶

Compute the auto- and cross- spectra of the given alm’s (either 1 alm or 3 alm).

Parameters :

alm : array or sequence of arrays

one array or a sequence of 3 arrays of identical size

mmax : int, optional

The maximum m for alm(s)

nspec : int, optional

The number of spectra to return if 3 alms were given: nspec==[0-6], in order TT,EE,BB,TE,TB,EB.
Returns :

cl : array or sequence of arrays

the power spectrum estimated from alm.
